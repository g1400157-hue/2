# 🖥 BK Server Manager

Утилита для массового управления серверами TRASSIR. Поддерживает проверку NTP, Web/Cloud статусов, мониторинг БД, POS-терминалов и управление пользователями с эталонами прав.

## 🚀 Быстрый старт
1. Установи [Python 3.8+](https://www.python.org/downloads/)
2. Скачай репозиторий или клонируй:
   ```bash
   git clone https://github.com/g1400157-hue/bk-server-manager.git
   cd bk-server-manager
